# Amioud

**Live preview:** [https://dentistici-hub.github.io/denis-website-amioud/](https://dentistici-hub.github.io/denis-website-amioud/)

Static marketing site for Amioud — niche hair-fragrance house (extrait de parfum per capelli, 20% concentration, Violet Collection). Built with Astro and deployed to GitHub Pages as a demo preview.

## About this site

- **Client:** Amioud (Aura Company S.r.l., VAT IT13285940014)
- **Industry:** Luxury niche perfumery / D2C beauty
- **Location:** Torino, Italy (Via Nicola Fabrizi 43)
- **Source site (for reference):** https://www.amioud.com/

## Build

```bash
npm install
npm run dev      # local preview at http://localhost:4321/denis-website-amioud/
npm run build    # static output in dist/
npm run preview  # serve built output
```

## Deploy

Automated via `.github/workflows/deploy.yml` on push to `main`. GitHub Pages serves from the workflow artifact (no `gh-pages` branch).

## Stack

- Astro 5 (static output, no runtime framework)
- Motion: GSAP + Lenis + Lottie via `Website/components/motion-primitives` (scroll-scrub, clip-mask-reveal, ken-burns, parallax, marquee, cutout)
- Type: Cormorant Garamond (display) + Inter (body), self-hosted via Fontsource
- Tokens: onyx #1a171b on cream #f3eee6, violet accent #5a3e8c, gold secondary #b89768

## Demo mode

This build is in demo mode:
- `<meta name="robots" content="noindex, nofollow">` on every page
- A demo banner at the bottom of every page
- No analytics, no booking backend, no Shopify write access

The `promote amioud` command in the Website builder pipeline flips this to production and prepares the site for a custom domain via Cloudflare.
